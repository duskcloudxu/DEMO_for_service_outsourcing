## 开发注意事项!important
### 1. 
图片不能用相对路径，否则会造成安卓无法读取的情况，统一放在
> asset/imgs/

目录下，填写src目录时用例为：
```
<img src="assets/imgs/myLawyer.png" margin-top="10px">
```
